# Jsuper7 Tier List Checker

เว็บแอปพลิเคชันสำหรับ **ตรวจสอบ Tier List ของ Jsuper7**
ผู้ใช้สามารถกรอก **หมายเลขโทรศัพท์** เพื่อตรวจสอบระดับ Tier และข้อมูลสถานะของตนเองได้

โปรเจกต์นี้พัฒนาด้วย **Next.js** และสร้างขึ้นด้วย `create-next-app`

---

## 🚀 คุณสมบัติของระบบ

* ตรวจสอบ Tier List ด้วยหมายเลขโทรศัพท์
* ใช้ Next.js API Routes สำหรับการดึงข้อมูล
* หน้าตาเว็บเรียบง่าย ใช้งานง่าย
* รองรับการ Deploy ได้ทันที

---

## 🛠 เทคโนโลยีที่ใช้

* **Next.js**
* **React**
* **Node.js**
* **JavaScript**
* **CSS**

## 📦 การเริ่มต้นใช้งาน

ติดตั้ง dependencies ก่อน

```bash
npm install
```

จากนั้นรันเซิร์ฟเวอร์สำหรับพัฒนา

```bash
npm run dev
```

เปิดเว็บเบราว์เซอร์แล้วไปที่

```
http://localhost:3000
```

คุณสามารถแก้ไขหน้าเว็บได้ที่ไฟล์

```
app/page.js
```

เมื่อมีการแก้ไขไฟล์ หน้าเว็บจะอัปเดตอัตโนมัติ

## 🌐 การ Deploy

วิธีที่ง่ายที่สุดในการ deploy โปรเจกต์ Next.js คือใช้ **Vercel**

ขั้นตอน

1. Push โปรเจกต์ขึ้น GitHub
2. เชื่อม repository กับ Vercel
3. Deploy ได้ทันที
